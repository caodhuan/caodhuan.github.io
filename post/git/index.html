Git

November 26, 2023

git 操作

1. 初始化仓库

git init

2. 查看当前仓库状态

git status

3. 提交代码

git add .
git commit -m "commit message"

4. 回退本地修改

git restore

5. 拉取代码

git fetch
git merge

6. 查看日志

git log

7. 检查更改

git blame "file name"

8. 查看分支

# 查看本地分支
git branch
# 查看所有分支
git branch -vv
# 切换分支
git switch branch name
